# ATM Secure PIN Entry Device (Academic Lab)

## Objective
Simulate a secure ATM PIN Entry Device (PED) architecture with:
- Secure PIN handling
- AES encryption
- Attempt limiting
- Tamper simulation
- Fault injection testability

## Phases
1. Core PIN module
2. Crypto module (AES + DUKPT simulation)
3. Tamper detection module
4. Secure boot simulation
5. Fault injection experiments
